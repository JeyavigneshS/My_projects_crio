PROBLEM TASK - PRIMARY
Use Case
In the realm of data analysis, it's common to encounter datasets with missing or inaccurate information. This often happens in sales data, where certain values might not be recorded due to various reasons. Addressing these missing values effectively is crucial to ensure the data is reliable and can be used for decision-making and forecasting.
Problem Description
Implement a function clean_sales_data that:

Accepts a pandas DataFrame containing sales data.
Identifies and processes columns with missing values according to specified criteria:
Fills missing values in the "Revenue" column with the mean revenue.
Replaces occurrences of "Unknown" in the "Region" column with "Not Specified".
Drops rows where the "Product" column is missing.
And returns the cleaned DataFrame.
Notes
Ensure that the input DataFrame has columns named "Revenue", "Region", and "Product".
The mean revenue should be calculated only from the non-missing values in the Revenue column.
