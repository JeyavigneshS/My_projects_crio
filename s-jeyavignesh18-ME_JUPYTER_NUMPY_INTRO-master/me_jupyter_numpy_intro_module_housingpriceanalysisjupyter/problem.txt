PROBLEM TASK - PRIMARY
Concept Recap
In statistics, measures of central tendency (mean, median, and mode) summarize a dataset with a single representative value.

`Mean`: The sum of all values divided by the number of values.
`Median`: The middle value in a sorted list of numbers. If there is an even number of values, it’s the average of the two middle ones.
`Mode`: The most frequent value in the dataset.

Outliers (extreme values) can distort the mean, making it less representative. The `median` is more robust to outliers.
<br>
Problem Description
A real estate analyst needs to find the typical home price in a neighborhood, but the dataset includes unusually high prices that might skew the result.

Write a function `find_central_tendency(prices)` that:
`Returns the mean` if there are `no outliers` (prices ≤ $1,000,000).
`Returns the median` if there are `outliers` (any price > $1,000,000).
`Returns the mode` if `all values are identical`.
If all values are unique, return `None` for mode.
Outlier Detection:
Consider a price `above $1,000,000` as an outlier.
If outliers are present, always return the `median`.
If no outliers, return the `mean` unless all values are identical.
