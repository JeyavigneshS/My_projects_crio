Task:
Load the Dataset:
Dataset URL : gitlab
Load the student data from the provided CSV file into a DataFrame. The dataset should include the columns: `Student`, `Math`, `Science`, and `English`.


Calculate Total Score:
Calculate the total score for each student by summing the scores across the Math, `Science`, and `English` columns.
Create a new column, `Total Score`, in the DataFrame to store these values.

Apply Ranking Methods:
For each ranking method (average, min, max, dense), apply the Pandas `rank()` function to rank the students based on their `Total Score`:
'average': Tied students will receive the average rank for their group.
'min': Tied students will receive the lowest possible rank in their group.
'max': Tied students will receive the highest possible rank in their group.
'dense': Tied students will receive consecutive ranks with no gaps.
Return the Updated DataFrame:
Update the DataFrame to include columns for each ranking method (e.g., `Rank_Avg`, `Rank_Min`, `Rank_Max`, `Rank_Dense`), and return the final DataFrame with the rankings for each student.
