PROBLEM TASK - PRIMARY
Use Case
In a data-driven world, it is common to collect information in a structured format like DataFrames. The task of creating a sample DataFrame can help in understanding the representation of tabular data and facilitate further data manipulation and analysis.
<br>
Problem Description
Implement a function create_sample_dataframe that:

Creates a DataFrame `df` containing sample data from a dictionary
```
{
        "Name": ["Alice", "Bob", "Charlie"],
        "Age": ["25", "30", "35"],
        "Grade": [85, 90, 75],
        "City": ["New York", "Boston", "Chicago"]
}
```
The function should use `dtypes` and print the information related to the datatypes present in the dataframe.
The function should return the constructed DataFrame.
<br>

Implement a function create_and_analyze_dataframe that:

Creates a DataFrame `df` representing a group of students with their name, age, grade, and city, as per the following dictionary
```
{
        "Name": ["Alice", "Bob", "Charlie"],
        "Age": ["25", "30", "35"],  # Strings
        "Grade": [85, 90, 75],      # Integers
        "City": ["New York", "Boston", "Chicago"]
}
```
Converts the 'Age' column from String to `int` type by using `astype`.
Computes the average age `avg_age` of the students.
Returns a tuple containing the DataFrame and the average age.
<br>
