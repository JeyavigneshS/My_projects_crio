PROBLEM TASK - PRIMARY
Use Case
In a data-driven environment, businesses often need to load product data from a file to analyze and manage inventory. This problem simulates the task of transferring data from an external source (Excel) to a structured format for further processing.
<br>
Problem Description
Implement a function load_products_excel that:

Accepts no arguments
Reads the data from an Excel file and returns a DataFrame containing the product data.
<br>

Implement a function load_products_excel_with_index that:

Accepts no arguments
Loads product data from the 'Products' sheet of a specified Excel file, using product_id as the index column.
Returns a DataFrame containing the product data.
<br>

Implement a function load_all_sheets that:

Loads multiple sheets i.e. the `Products` and the `Merchants` sheets from a specified Excel file.
Returns a dictionary such that the keys of the dictionary should be the names of the sheets, and the values should be the corresponding DataFrames.
