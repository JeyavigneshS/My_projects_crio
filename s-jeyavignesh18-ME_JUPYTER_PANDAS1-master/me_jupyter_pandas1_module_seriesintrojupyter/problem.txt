PROBLEM TASK - PRIMARY
Use Case
In a retail business, keeping track of sales data on a monthly basis is crucial for understanding trends and performance. Having this data organized and easily accessible allows businesses to make informed decisions.
<br>
Problem Description
Implement a function create_monthly_sales_series that:

Creates and returns a Pandas Series containing monthly sales data.
The sales data should represent sales figures for the first six months of the year.
Make sure to use appropriate indexing for the Series.
```
["January", "February", "March", "April", "May", "June"]
```
The function should return a Series indexed by month, with the sales values mentioned in the below array:
```
[200, 300, 250, 400, 350, 500]
```
<br>

Implement a function get_sales_for_month that:

Accepts an argument sales, a pandas Series containing monthly sales data indexed by month names.
Returns the sales value for the month of March.
<br>

Implement a function update_sales_for_month that:

Accepts an argument sales, a pandas Series containing monthly sales data indexed by month names.
Updates the sales amount for the month of June to a fixed amount of 550.
Returns the updated sales series.
<br>

Implement a function increase_sales_by_percentage that:

Accepts a series of monthly sales data as input
Increases each sales figure by 10%
And returns the updated series of sales data
<br>
