Use Case
In a data analysis process, it's common to load data files in various formats while selectively choosing which rows to include. In particular, we might need to skip specific rows that contain metadata or unwanted information.
<br>
Problem Description
Implement a function load_exam_review_skiprows that:

Loads exam review data from a CSV file using the custom separator > and skips the first 2 rows.
Returns the loaded data as a DataFrame.

Loads exam review data from a CSV file.
The data should use a custom separator '>'.
